# 🎬 Anime World Archive

A cinematic, high-performance web archive dedicated to the history of anime, manga, and the legendary filmography of **Studio Ghibli**. This project focuses on a professional user experience with a dark, industrial UI.

## 🚀 Live Demo
[View the Live Site here!](https://github.com/thoon-yamone-naing1/anime-world/)

## ✨ Key Features
* **Clean UI Practice:** A dark-themed, cinematic interface built while learning CSS3 layout techniques.
* **Studio Ghibli Deep Dive:** Explores the origins of the name "Ghibli" (derived from the Italian word for a hot desert wind) and its evolution.
* **Comprehensive Archive:** Covers anime history, manga milestones, and industry awards.
* **High-Fidelity Assets:** Curated imagery from classics like *Grave of the Fireflies* and modern hits like *Frieren: Beyond Journey's End*.

## 🛠️ Technical Stack
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and a "Cyber-Industrial" aesthetic.
